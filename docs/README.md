1. 所有文档使用 Markdown 格式
2. 文档标题使用中文,代码示例使用英文
3. 重要变更需更新 [CHANGELOG.md](CHANGELOG.md)
4. 新增文档需更新本索引文件

### 文档分类规则
- **architecture/**: 系统设计、技术架构相关
- **guides/**: 操作指南、实施手册
- **analytics/**: 分析方案、指标定义
